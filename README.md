# Erica_Talk
2019 Opensource Lecture Project

## 실행 하는 법

#### node.js 와 npm 이 설치되어 있는 걸 전제로 합니다.

~~~
clone https://github.com/ArcheSeraph/Erica_Talk.git
// clone 한 경로로 이동
npm i
npm run build

npm test // 방법 1
//방법 2
// 리눅스는 packaging 된 파일 경로로 이동
./Erica_Talk
// 윈도우와 맥은 실행파일을 실행
~~~

### _주의_
* _로컬 사용자가 서버를 열어야 사용이 가능합니다._
